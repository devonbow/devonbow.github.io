Progressive Web Application (PWA)

Progressive Web Applications combine different technologies to make powerful web applications. PWA's tend to be reliable, fast and engaging. When the app is launched, the PWA will load instantly, regardless of the network state. Users can install PWA's, and those will live on the users' home screens without the need for any type of app store. These applications offer a very rich experience with help from a web-app manifest file and can re-engage users with web-push notifications.

When the PWMA criteria is met, Chrome will prompt users to add high-quality PWA's to their home screens. PWA's make use of service workers, which help send data faster, which will then allow users to complete tasks faster. Depending on how you use the push notifications, you might increase engagement, which might directly or indirectly increase sales (if you are selling something). This means you will have the benefit of improved conversions.
